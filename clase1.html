HTML: Hyypertext Markup Language

¿Que es HTML

Es un lenguaje compueto por etiquetas, las cuales describen elementos. El navegador web es el encargado de interpretar el código y mostrarlo de forma comprensible a nuestros usuarios.

Justo con CSS y javacript, HTML es parte de la triada de lenguajesque definene a neustra Web.

Elemtenos HTML y semántica

Son bloques de contruicción para la maqeuta de nuestra página Eb. Los elementos HTML nos permiten incorporar texto, imágenmes, vinculos, entro otros. HTML también proporciona un contexto semántico de elemetnos el cual describe su propósito, como por ejemplo títulos, párrafos, listas, encales, citass y otros.

Hojas de estilo en cascada
